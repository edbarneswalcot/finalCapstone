Project Name: US Arrests Analysis

Description: This project aims to analyze the United States Arrests data. The results of this analysis present levels of violent crime across states and 
regions in the US. This analysis can help US citizens better understand what states are the most violent and how crime is spread across the country.

1. Installation:
This project is hosted on GitHub, so the best way to access it is by forking the reposity and cloning a copy to your local machine.

2. Usage:
After you have a copy of the project on your local machine, you can open it in Jupyter Notebook. You will then be able to view the analysis. Also, you can 
modify the dataset and re-run the analysis.
